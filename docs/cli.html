Elpida CLI Documentation

Default Usage

elpida-cli

Example Usage with parameters

elpida-cli --format=json --upload ...

Options

  • -v, --version: Prints the version and exits.
  • -h, --help: Prints this help and exits.
  • --format=FORMAT: The result output format. Accepted values: default, json (default).
  • --delay-seconds=SECONDS: The delay in seconds between runs (Default is 0).
  • --upload: Enable uploading the result to elpida website (disabled by default).
  • --generate-report=[OUTPUT_FILE]: Whether to generate an HTML report. If no path is specified, it is output to 'Elpida-Report.html'.
  • --iterations=N: How many iterations to run (Default is 1).
  • --benchmarks-directory=DIRECTORY: Where to find the benchmarks (Default is './Benchmarks').
  • --output=FILE: Output the results on a file instead of STDOUT.
  • --quiet: Output only the results on STDOUT
