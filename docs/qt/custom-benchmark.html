Custom Benchmark

This tab is about custom benchmarking. Here you can run individual benchmarks with specific configuration.

Available benchmarks

A tree with the available benchmarks you can run. Here expand and select which one you want to run. Once selected, its configuration will appear on the **Configuration** pane.

Configuration

Here is the available configuration values you can assign to the selected benchmark. Depending on the benchmark, the configuration values and types changes.

Execution

This controls the execution of the benchmark. See Advanced Benchmarking for more information on the configuration.

You can press the Start button to start the benchmark and can press Cancel to cancel current execution.

Results

Here are listed the results of the past executions.
