Benchmark

This tab is about general benchmarking your CPU. This will run a batch of pre-configured benchmarks to determine its speed compared to other CPUs on the same benchmarks.

Score

This group shows the scores for the most recent benchmark run in this session.

  • The first number with the large letters is the total score. Next to it is the difference with the previous run.
  • Single thread: The single thread score for the latest run. Next to it is the difference with the previous run.
  • Multi thread: The multi thread score for the latest run. Next to it is the difference with the previous run.

Execution

This group has the controls for running the benchmark.

  • Current benchmark: N/A: This shows the current benchmark name that is running.
  • Run it: Here you can set how many times the benchmark should run in succession.
  • Delay between runs: The amount of seconds to delay between runs when you selected multiple runs via the **Run it** field.
  • Upload results anonymously to elpida website: When checked and the benchmarks are finished, the results will be uploaded to elpida public website to be counted to statistics for this CPU.
  • Open result in browser after upload: If chosen to upload the results, after upload completes the result will be opened in your default browser in the public page of the first result of the batch that was run.
  • Generate HTML report when complete: When checked and the benchmarks are complete, an HTML report will be generated on disk with the results and statistics. Also, the HTML page will open in your default browser after generated.
  • Start: This button will start the benchmarking. When the benchmarks run, this button will change to **Cancel**, where clicking it will cancel the current benchmark execution.
  • Estimated execution time: The amount of time the benchmark will take to complete. This changes depending on the amount of **Run it** and **Delay between runs** you set.

Results history

This contains the results of runs you performed during this session. Each line is expandable and shows the individual benchmark results as well as the delta with the previous execution.
